Raspberry Pi Pico

The Raspberry Pi Pico is a small microcontroller based board that uses minescule amounts of power to compleate a task. The Raspberry Pi Pico has two core meaning you can program it to run two parrallel tasks and it can be programmed in micropython (Subset on python) and even in C++. The Raspberry Pi Pico can be used for many things like to data logging, small to mediumm sized robots, analog sensing, etc. It is a small scale cheap and simple board.
